## Responsive Breakpoints

### Desktop (> 1024px)
- Sidebar: Fixed position, full height
- Default: Open (288px)
- Toggle available

### Tablet (768px - 1024px)
- Sidebar: Fixed position, full height
- Default: Collapsed (80px)
- Toggle available

### Mobile (< 768px) - Future
- Sidebar: Overlay mode
- Default: Hidden
- Hamburger menu button
- Backdrop blur when open
